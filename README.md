# HortiFácil

Projeto acadêmico de sistema de gestão de hortifrúti desenvolvido em Java.

## Descrição
Aplicação desktop com foco em backend, utilizando Java, JavaFX e MySQL.
O sistema possui módulos para **Administrador** e **Cliente**, com:
- Controle de estoque (FIFO)
- Cadastro e gerenciamento de produtos
- Carrinho de compras e pedidos
- Assinaturas e relatórios

## Tecnologias
- Java
- JavaFX
- MySQL
- JDBC
- Maven

## Arquitetura
- Controller
- Service
- DAO
- Model

## Como executar
1. Configurar o banco de dados MySQL
2. Ajustar credenciais no projeto
3. Executar `Main.java`
